# twinpeaks-pokemon (prototype)

Super-minimal prototype: a top-down “Pokémon DS-ish” feel with a rectangular map and arrow-key movement.

## Run

Open `index.html` in a browser.

If your browser blocks local file features, use any tiny static server, for example:

```bash
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.
